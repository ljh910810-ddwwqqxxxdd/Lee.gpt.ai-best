# AI 학습 도우미 프로젝트

## 소개

이 프로젝트는 사용자가 입력한 정보를 바탕으로 학습 내용을 정리하고, 필요한 결과를 쉽게 확인할 수 있도록 도와주는 간단한 AI 기반 도구입니다.

복잡한 자료를 직접 하나씩 정리하지 않아도 핵심 내용을 요약하고, 주요 정보를 구조화하여 사용자가 더 효율적으로 학습하거나 작업할 수 있도록 돕는 것을 목표로 합니다.

## 주요 기능

- 입력된 텍스트 또는 데이터를 분석하여 핵심 내용을 정리합니다.
- 사용자가 이해하기 쉽도록 결과를 항목별로 구조화합니다.
- 반복적인 정리 작업을 줄여 작업 시간을 절약할 수 있습니다.
- 간단한 실행 방식으로 누구나 쉽게 사용할 수 있습니다.
